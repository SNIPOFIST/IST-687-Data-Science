# Lecture Materials and Learning Activities

## Overview

This folder contains lecture materials, practice exercises, and learning activities from the IST-687 Data Science course. These materials demonstrate the learning process and skill development throughout the semester.

## Contents

### Learning Exercises
- `Test_data.Rmd` - Data exploration and testing exercises

## Course Learning Objectives Covered

### Data Science Fundamentals
- Data collection and preprocessing techniques
- Statistical analysis and hypothesis testing
- Data visualization best practices
- Reproducible research methodologies

### Technical Skills Development
- R programming proficiency
- Data manipulation with dplyr and tidyr
- Advanced visualization with ggplot2
- Interactive applications with Shiny

### Analytical Techniques
- Exploratory data analysis
- Time series analysis
- Predictive modeling
- Statistical inference

## Learning Progression

The materials in this folder show the progression from basic data exploration to advanced analytical techniques, culminating in the comprehensive project showcase.

These materials serve as:
- Reference for data science techniques
- Examples of best practices in R programming
- Documentation of learning journey
- Foundation for advanced project work
